Connect to Google Sheets

This dashboard reads and writes directly to your spreadsheet. You need to set up the Apps Script bridge once — takes about 5 minutes.

  1. Open your Google Sheet
  2. Click Extensions → Apps Script
  3. Delete any existing code, paste in Code.gs
  4. Save, then run setupSheets() once (sets up headers + seed data)
  5. Click Deploy → New deployment → Web app
  6. Set Execute as: Me — Who has access: Anyone
  7. Copy the deployment URL and paste it below
